========= EXPLANATION =============

hello this will review all password combos for an aps student if you do a full
check it will take approximately 5.5 hours to check every combo assuming it is
checking at a speed of 0.2 checks a second

------------ USAGE ----------

Once you run the program you will need to enter the range of passwords you want
to check this needs to be 5 character long and only use numbers. For example
if you want to check all passwords between 70000 and 60000 you would enter
and who you want to find
60000
70000
thudu2683@xxxxx.org

when this is done it will turn off and log the answer in password.txt
